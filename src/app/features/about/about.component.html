About This Project

nG Portable UI showcases a portable Angular 20 design system built with standalone components, Tailwind utility theming, and a vendor abstraction layer (Web Awesome, AG Grid planned). This page will evolve with project history, goals, and architecture notes.

Philosophy

Single swap point for vendors. Neutral API. CSS variable driven theming.

Stack

  • Angular 20 standalone
  • Tailwind v3
  • Web Awesome
  • AG Grid (soon)

Status

Core ds-* wrappers implemented. Grid + theming tokens next.
